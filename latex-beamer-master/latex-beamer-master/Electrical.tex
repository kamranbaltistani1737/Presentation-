    \frametitle{Electrical Design}
\begin{columns}[T]   % <-- important (Top alignment)

\column{0.6\textwidth}
    Main electrical component used to design this project are.
    \begin{itemize}
        \item[$\diamond$] Two microcontrollers
        \begin{itemize}[label=$\checkmark$]
        \item STM32F401: Primary Controller with extensive GPIO.
	    \item DFRobot Beetle ESP32 C6: For wireless communication with ground station.
        \end{itemize}
        \item[$\diamond$] Motor Driver : Receives PWM control signals from the STM32 and uses the high-power battery supply to generate controlled three-phase outputs for regulating the motor speed. 
        \item[$\diamond$] 3x T-Motor GB2208:Three phase motors act as reaction wheels, producing controlled torques to regulate the FloatSat’s roll, pitch, and yaw orientation.
    \end{itemize}
